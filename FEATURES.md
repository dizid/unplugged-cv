# unplugged.cv Features

## Core Features

### CV Generation
- **AI-Powered Generation**: Paste unstructured career info, get a polished CV
- **Real-time Streaming**: Watch your CV generate live
- **Markdown Output**: Clean, portable format
- **PDF Export**: Print-ready PDFs with professional styling
- **Copy to Clipboard**: One-click markdown copy

### Job Analysis
- **Job Description Parser**: Paste any job posting for structured analysis
- **Requirements Extraction**: Must-have vs nice-to-have skills
- **Red Flag Detection**: Identifies problematic job postings
- **Seniority Signals**: Understand role level expectations
- **Salary Detection**: Extracts mentioned compensation

### Cover Letters
- **AI-Generated**: Tailored to job description + your CV
- **Job-Specific**: Highlights relevant experience
- **PDF Export**: Matching professional styling

### Import Options
- **Text Paste**: Raw career info, LinkedIn copy-paste
- **URL Import**: Fetch content from career pages
- **Google Docs**: Import directly from Google Docs

### Publishing
- **Public CV URLs**: Shareable links (e.g., `unplugged.cv/cv/yourname`)
- **Custom Slugs**: Choose your URL path
- **Always Updated**: Edit and republish anytime

## Technical Features

- **Responsive Design**: Works on mobile, tablet, desktop
